How to use the API

Just supply the API endpoint with url and word parameters.

url should be a full URL, like http://virtusize.jp.

word parameter can be anything. It can contain a alphabets and numbers.

Your query should look like this.

https://{{ request.get_host }}/wordcount?url=http://virtusize.jp&word=fit
